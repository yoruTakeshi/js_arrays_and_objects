# Glossário (apenas o mais importante)

## Array:
- Basicamente uma lista
- Método simples e rápido, sem estrutura
- [MDN](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript/Reference/Global_Objects/Array)

## Objeto:
- É como se fosse uma variável com várias sub-variáveis
- Método com estrutura, mais detalhado
- Pode ser usado como uma lista com várias estruturas detalhadas dentro
- [MDN](https://developer.mozilla.org/pt-BR/docs/Learn/JavaScript/Objects/Basics)

## document.querySelector():
- Seleciona algum elemento do HTML, seja por nome do elemento, classe ou id
- [W3Schools](https://www.w3schools.com/jsref/met_document_queryselector.asp)

## Método .length:
- Simplesmente coloque-o na frente de algo compatível e você recebe um número que representa a quantidade de coisas que você tem em algo

## Método .innerHTML:
- Seleciona DENTRO de um elemento HTML
- [MDN](https://developer.mozilla.org/pt-BR/docs/Web/API/Element/innerHTML)
